# InsightIQ Business Intelligence Dashboard

> **Transforming Business Data into Actionable Insights**

InsightIQ is a comprehensive, production-ready Business Intelligence Dashboard built for businesses of all sizes. It provides real-time analytics, KPI tracking, customer management, inventory control, financial reporting, and more.

## 🌟 Features

### Core Features
- **Real-time Dashboard** - Interactive KPI cards, charts, and data visualization
- **Analytics** - Advanced analytics with filtering, date ranges, and export options
- **Sales Management** - Track orders, revenue, and sales performance
- **Customer Management** - Full CRUD with segmentation and analytics
- **Product & Inventory** - Stock management, alerts, and movements tracking
- **Employee Management** - Department, attendance, and performance tracking
- **Financial Management** - Invoices, expenses, profit & loss, and budgeting
- **Reports** - Exportable reports in PDF, Excel, CSV, and print formats
- **Multi-role Support** - Super Admin, Admin, Manager, Staff, Viewer
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Dark/Light Mode** - Full theme support with toggle
- **RESTful API** - Complete API with authentication and CRUD endpoints

### Security Features
- Password hashing with bcrypt
- CSRF protection
- SQL injection protection (PDO)
- XSS protection
- Rate limiting
- Session security
- Audit logging
- Two-factor authentication (structure ready)
- Secure file uploads

### Performance Features
- Optimized database queries
- Caching structure
- Lazy loading
- Minified assets
- Pagination for large datasets
- Efficient DOM manipulation
